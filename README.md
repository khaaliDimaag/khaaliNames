# khaaliNames
> An onchain library for generating unique names for user management

`khaaliNames` is an onchain library primarily developed for khaaliDimaag projects to manage user 
registrations and generating unique usernames inspired by the 
[unique-names-generator](https://www.npmjs.com/package/unique-names-generator) npm package. The 
goal is to have an onchain library that projects and dApps can call to generate new usernames for 
their users and manage them using ENS Subnames as 7579 Modules.
